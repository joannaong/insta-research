instagram API

http://instagram.com/developer/

Instagram API uses OAuth 2.0, requests must be made over SSL (https:// not http://).

We can get these from the API

Users

  • get user-id
  • get user's feed
  • get user's media (both video and image)
  • get user likes
  • search for a user by name

Relationships

  • get user's followers
  • get user's following
  • get list of users who have requested this user's permission to follow
  • get relationship with another user
  • modify relationship between current user and target user

Media

  • get media id
  • Search for media, both image and video types, in a given area (latitude, timestamp, longitude, distance)
  • Get a list of what media is most popular at the moment (both image and video types)

Comments

  • get full list of comments
  • create a comment
  • delete a comment

Likes

  • get list of users who liked the media
  • set a like on media by the currently authenticated user
  • eemove a like on this media by the currently authenticated user

#Tags

  • get information about a tag object
  • get a list of recently tagged media.
  • search tags by name

Location

  • get location is
  • get a list of recent media objects from a given location
  • search for a location by geographic coordinate.

Geography

  • get recent media from a geography subscription that you created

TL;DR

  • We CANNOT post media (video, image) on a user's feed / timeline.
  • We CAN add / delete comments, add / count likes, get user's followers, get media using tags.
